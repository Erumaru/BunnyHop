----- HELP -----
The way commands are displayed:
<parameter> is a required parameter
(parameter) is a optional parameter
-----
List of console commands:
<b>help</b> <i>This</i>
<b>quit</b> <i>Closes the game</i>
<b>forcequit</b> <i>Closes the game no matter what</i>
<b>connect <ip> <port> (password)</b> <i>Connects to a multiplayer server</i>
<b>disconnect</b> <i>Disconnects from a server</i>
<b>newserver (password)</b> <i>Creates a new server and connects the client to it</i>
<b>stopserver</b> <i>Stops the server if you are running it</i>
<b>playdemo <demoname></b> <i>Plays back a demo with the given file name. Demos are read from the "Velocity 0.1.X_Data" folder.</i>
<b>move_friction (new friction multiplier)</b> <i>When the player touches ground, the speed will get multiplied with this.</i>
<b>move_speed (new input speed)</b> <i>Player movement input will get multiplied with this</i>
<b>move_maxspeed (new max speed)</b> <i>Player speed limit. Once reached, only strafing can increase the speed further.</i>
<b>move_airspeed (new air speed)</b> <i>Air control (influences air strafing a lot)</i>
<b>move_jumpheight (new jump height)</b> <i>The force with which the player gets pushed upwards when jumping</i>
<b>move_gravity (new gravity)</b> <i>Global gravity</i>
-----
